# sform

1. Run docker containers with
  `$ docker-compose up -d`
2. Enter to the PHP container 
  `$ docker exec -it sform_php_1 bash`
3. Enter to the project folder
  `$ cd ./html/sform`
4. Run composer install
  `$ composer install`
5. Run npm install
  `$ npm install`
6. Build in dev mode
  `$ npm run dev --watch`   
7. Open in browser
   `localhost:8080`
